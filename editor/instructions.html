The purpose of this application is to assist you in taking notes, it will seamlessly help in writing advanced equations to formatting your notes in an organized and clean fashion. Below is a list of our introductory tools that will make your note taking process, via computer and/or touch screen device, a little more easier.

General Instructions

To use any of our formatting shortcuts please start with the `\` (back slash) followed by the desired shortcut

General Formatting Shortcuts

  • `1` - Large Title
  • `2` - Medium Title
  • `3` - Small Title
  • `i` - Italic
  • `b` - Bold
  • `u` - Underline
  • `_` - SubScript (Note: a `\` back slash is not needed for this shortcut)
  • '^' - SuperScript (Note: a `\` back slash is not needed for this shortcut)

Equation Shortcuts

To activate equation mode please use the `eq` shortcut followed by any of the following commands

  • `^` - for superscript formatting with items such as exponent (x2) Note: a `\` back slash is not needed for this shortcut
  • '_' - for subscript formatting with items such as sequences (x2) Note: a `\` back slash is not needed for this shortcut
  • `int` - for the integral symbol
